Este texto está flotando a la derecha

Este es el flujo Normal de un Documento; de arriba a abajo, de izquierda a derecha. Cuando se agrega el texto flotante, se mueve a la esquina superior derecha del elemento que contiene, en este caso el div. El texto normal fluye alrededor del texto flotante.
